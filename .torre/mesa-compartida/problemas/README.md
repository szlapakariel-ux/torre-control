# Problemas planteados por Ariel

Acá se cargan problemas en lenguaje natural antes de convertirlos en órdenes.

Cada problema debe tener estos campos:

- PROYECTO_FUNCIONAL
- REPO_TECNICO
- PROBLEMA_DE_ARIEL

Sin PROYECTO_FUNCIONAL y REPO_TECNICO, no se procesa.

Formato sugerido de archivo: <id>-<slug>.md

Ejemplo: 001-secretaria-notas-whatsapp.md
