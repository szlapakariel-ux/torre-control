# Órdenes derivadas por Torre

Acá se guardan las órdenes que Torre genera a partir de los problemas de Ariel.

Cada orden debe indicar:

- PROYECTO_FUNCIONAL
- REPO_TECNICO
- RESPONSABLE_ACTUAL
- ARCHIVOS_A_TOCAR
- ARCHIVOS_PROHIBIDOS
- CRITERIO_DE_EXITO

La orden debe ser suficientemente clara para que un agente pueda ejecutarla sin pedir contexto adicional.
